В рамках задания была создана и размещена на GitHub сложная блок-схема алгоритма. Схема включает в себя условные операторы (if/else) для ветвления логики и операторы вывода (print) для отображения результатов.


```mermaid
graph TD
    Start([Начало]) --> Input[/"Введите логин и пароль"/];
    Input --> Check{Пароль верен?};
    Check -- Да --> PrintSuccess[/"Вывести: 'Доступ разрешён'"/];
    Check -- Нет --> PrintError[/"Вывести: 'Ошибка входа'"/];
    PrintSuccess --> Log[/"Записать лог входа"/];
    PrintError --> Log;
    Log --> Finish([Конец]);
```
